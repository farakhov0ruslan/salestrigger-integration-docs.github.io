SalesTrigger — Quick Start Guide

Follow the steps below to set up the app, create a campaign, and run modules. Screenshots are referenced as “Photo N”.

1) Registration (Photo 1)

  1. Create an account / sign in to the platform.
  2. Navigate to your profile or organization settings.
Photo 1 — Registration

2) Create and Copy an API Key (Photos 1–3)

  1. Open API TOKENS (or “Developers / Keys”).
  2. Create a new key, set a recognizable name and scope if required.
  3. Copy the generated key and store it securely.
Photo 1 — Keys page
Photo 2 — Create key
Photo 3 — Copy key

3) Connect the Integration (Photo 1)

Add a new Connection and paste your API key into the required field. Save the connection and verify it.

Photo 4 — Connection setup

4) Create a New API Campaign (Photos 1–3)

  1. Open the platform and create a new API campaign.
  2. Fill in the campaign name and basic settings.
  3. Save and confirm that the campaign appears in the list.
Photo 1 — Campaign in menu
Photo 2 — New Campaign button
Photo 3 — Crete campaign Menu

5) Configure Sequence Settings

Open Sequence and set when each action should be performed (delays/intervals). Save the configuration.

Photo 1 — Sequence setup

6) Module: Add Lead to an API Campaign

  • Select the campaign you just created.
  • Paste a valid LinkedIn profile URL of the lead you want to add.
  • Enter the messages you want to send. If all messages are empty, the module will send only a connection request.
Photo 1 — select campaign, add url

7) Module: API Call

Use this universal module to call any supported endpoint. For details on paths, parameters, and responses, see the documentation below.
